# Streaming Data Analysis

- Info from Video Streaming services either in india or out of india.
- Like Netflix, Amazon Prime Video, JioHotstar, Hulu, Parmount, etc.

## Dataset Details (Open Source)
- Data source : you are expected to use and download relevant dataset from open data source like Kaggle, UCI etc.]
- Data Type: Structured CSV dataset
- Domain: Video Streaming, streaming services.

This dataset is ideal for:
- EDA exploratory data analysis (Univariate, Bivariate, Multivariate)
- Handling missing values and outliers
- Regression analysis
- Introduction to supervised learning
